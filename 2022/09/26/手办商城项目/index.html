git操作

解决一次上传文件数量限制

前提：安装git工具，仓库已创建好

创建一个新文件夹，在该文件夹右击，选择:Git Bash here

git clone 仓库地址 （仓库地址为gitee网址加仓库名)

克隆完成后，该文件夹下会出现仓库文件夹，将需要上传的文件以及文件夹移到到该生成的仓库文件夹中

在仓库文件夹中右击，选择：Git Bash here

git add . 提交所有文件到暂存区

git commit -m ‘描述文字’ 提交到仓库

git push推送到远程库
